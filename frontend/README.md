# AI Provident Fund Navigator - Frontend

Next.js 15 Dashboard for Stock Market Prediction API.

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

Dashboard available at: http://localhost:3000

## Requirements

Make sure the Python backend is running at http://localhost:8000

## Features

- Real-time market prediction display (Bullish/Bearish)
- Confidence gauge visualization
- Portfolio allocation recommendation
- Backtest performance chart
- Technical indicators panel
